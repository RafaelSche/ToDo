Commands: create, delete, list, view
Arguments:
-i, --id (not useable for the command create)
-n, --name
-desc, --description
-dc, --date_created (not useable for the command create)
-dd, --date_deadline
-t, --tags

Examples:
>>>create -n "Name of item" -desc "Description of item" -t tag1 tag2 -dd 2019-02-16 # create some items
>>>create -n "Name of another item" -desc "Description of those" -t tag1
>>>create -n "Only name in this item"
>>>list                                        # list them with their attributes including ids
>>>list -t tag1                                         # list items with tag1
>>>view -i 5c6018759303fc1e2a04e727                                        # view one concrete item idetified by an id
>>>delete -i 5c6018759303fc1e2a04e727                                         # delete it
>>>delete -t tag1                                          # delete all with tag1
>>>delete -n "Name of item"                                         # delete all with name "Name of item"
>>>delete                                         # delete all
